MAX、MIN、AVG

5、求某一列值的最大值
This is a tip 
MAX 函数返回一列中的最大值。NULL 值不包括在计算中。
注释：MIN 和 MAX 也可用于文本列，以获得按字母顺序排列的最高或最低值。

1
2
3

SELECT 
	MAX(高考分数) as 高考最高分数 
FROM 学生表
6、求某一列值的最小值
1
2
3

SELECT 
	MIN(高考分数) as 高考最低分数 
FROM 学生表
7、求某一列值的平均值
1
2
3

SELECT  
	AVG(高考分数)  as  高考最高分数

MID

8、MID() 函数 ：用于从文本字段中提取字符。
1

SELECT MID(列名,开始截取位置,结束截取位置) FROM table_name
This is a tip 
SELECT MID(列名,1,3) FROM table_name
3不写，默认返回剩余的所有字段

1

1

三、SQL语言多表关联查询

一、外连接

外连接可分为：左连接、右连接、完全外连接。

1

select * from A表 left join B表 on A表条件 = B表条件
This is a tip 
左外连接包含left join左表所有行，如果左表中某行在右表没有匹配，则结果中对应行右表的部分全部为空(NULL)

1

select * from A表 right join B表 on A表条件 = B表条件
This is a tip 
右外连接包含right join右表所有行，如果左表中某行在右表没有匹配，则结果中对应左表的部分全部为空(NULL)。

1

select * from A表 full join B表 on A表条件 = B表条件
This is a tip 
完全外连接包含full join左右两表中所有的行，如果右表中某行在左表中没有匹配，则结果中对应行右表的部分全部为空(NULL)，如果左表中某行在右表中没有匹配，则结果中对应行左表的部分全部为空(NULL)。

二、内连接

join 或 inner join

1

select * from A表 join B表 on A表条件 = B表条件
This is a tip
inner join 是比较运算符，只返回符合条件的行。
